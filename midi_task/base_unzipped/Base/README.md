# 🎵 原始项目：MIDI 简易乐谱语言

## 项目结构

```
Base/
├── cjpm.toml           # 项目配置
├── README.md           # 本文件
├── src/
│   ├── main.cj         # 入口 + 测试套件
│   ├── midi_event.cj   # MIDI 事件类型定义
│   ├── parser.cj       # 文本乐谱解析器
│   ├── encoder.cj      # MIDI 二进制编码器
│   └── verifier.cj     # MIDI 文件验证工具
├── test.txt            # 测试乐谱
```

## 构建运行

```bash
cjpm run --run-args ./test.txt
```

执行后将在当前目录看到生成的`test.mid`文件，可以播放试听效果。

## 简易乐谱语法规则

### 1 总体结构

```
[指令行]     ← TEMPO / INSTRUMENT / TRACK 等全局设置
[音符行]     ← 由音符和休止符组成的旋律序列
[注释行]     ← 以 // 开头的注释，解析时忽略
[空行]       ← 忽略
```

### 2 指令

| 指令       | 格式                | 说明                             | 示例                |
| ---------- | ------------------- | -------------------------------- | ------------------- |
| TEMPO      | `TEMPO <bpm>`       | 设置速度（每分钟拍数），默认 120 | `TEMPO 140`         |
| INSTRUMENT | `INSTRUMENT <name>` | 选择 General MIDI 乐器           | `INSTRUMENT violin` |
| TRACK      | `TRACK <name>`      | 开始新轨道（多轨模式）           | `TRACK melody`      |

### 3 音符格式

```
音符格式: <音名>[升降号]<八度> <时值>
```

| 元素   | 取值                                              | 说明                     |
| ------ | ------------------------------------------------- | ------------------------ |
| 音名   | `C` `D` `E` `F` `G` `A` `B`                       | 7 个基本音名             |
| 升降号 | `#`（升半音）、`b`（降半音）                      | 可选，默认无             |
| 八度   | `0` - `8`                                         | 必填，中央 C 在第 4 八度 |
| 时值   | `1`=全音符 `2`=二分 `4`=四分 `8`=八分 `16`=十六分 | 必填                     |

**音符示例**：

- `C4 4` → 中央 C，四分音符（MIDI 音高 60，480 ticks）
- `F#5 8` → 高八度升 F，八分音符（MIDI 音高 78，240 ticks）
- `Bb3 2` → 低八度降 B，二分音符（MIDI 音高 58，960 ticks）

### 4 休止符

```
R <时值>
```

休止符表示静默，不生成 Note On/Off 事件，仅产生对应时长的延时。

- `R 4` → 四分休止符（480 ticks 的静默）
- `R 2` → 二分休止符（960 ticks 的静默）

### 5 和弦

```
[<音符1> <音符2> ...] <时值>
```

方括号内的音符同时发声，共享同一时值。

- `[C4 E4 G4] 2` → C 大三和弦，二分音符时值
- `[D4 F#4 A4] 4` → D 大三和弦，四分音符时值

### 6 注释

```
// 这是注释，整行忽略
C4 4  D4 4  // 行尾注释也支持
```

### 7 使用示例

```
// 小星星
TEMPO 120
INSTRUMENT piano

// 第一段
C4 4  C4 4  G4 4  G4 4
A4 4  A4 4  G4 2

// 第二段
F4 4  F4 4  E4 4  E4 4
D4 4  D4 4  C4 2
```
